# Workout Timer

A simple, customizable workout timer application to help you stay on track with your fitness goals.

## Features

- **Customizable Workouts:** Create your own workout routines by adding exercises and setting their duration.
- **Reorder Exercises:** Easily rearrange the order of exercises in your workout.
- **Timer View:** A clear and easy-to-use timer to guide you through your workout.
- **Clean Interface:** A simple and intuitive user interface.

## Tech Stack

- React
- TypeScript
- Vite
